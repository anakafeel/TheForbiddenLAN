---
title: Local Dev Bootstrap
description: One-command setup for local development
---

Run this command once to bootstrap the monorepo locally:

\`\`\`bash
pnpm setup:local
\`\`\`

This command enables Corepack, activates pinned `pnpm@10.23.0`, installs workspace dependencies, and performs Android environment sanity checks.

For more details, see [docs/LOCAL_DEV_BOOTSTRAP.md](/docs) in the repo.
